# Super Trunfo - Cidades Brasileiras #

##  Descrição do Projeto

Este é um jogo de Super Trunfo desenvolvido em linguagem C como parte do desafio "Nível Aventureiro" do curso. O programa permite ao usuário cadastrar duas cartas representando cidades brasileiras e comparar seus atributos em uma batalha interativa.

**Aluno:** Felipe de Lima Rodrigues  
**Disciplina:** Programação em C  
**Nível:** Aventureiro - Interatividade com Menu

---

## Funcionalidades

- Cadastro interativo de 2 cartas de cidades
- Cálculo automático de densidade populacional
- Cálculo automático de PIB per capita
- Cálculo automático de Super Poder (atributo especial)
- Menu interativo para escolha do atributo de batalha
- Comparação com lógica if/else para determinar vencedor
- Regra especial: densidade populacional (menor vence)
- Detecção de empate
